## Overview

An IMDb-like platform for discovering and searching new games, powered by Twitch's IGDB API.

## Key Features

- **Advanced Search Functionality**: Allows users to search for games based on multiple filters such as genre, platform, release date, and more, providing a tailored discovery experience.  
- **Dynamic Genre and Platform Filters**: Enables seamless browsing of games by genre or platform with intuitive filter options.  
- **Integrated IGDB API**: Powered by Twitch's IGDB API to fetch up-to-date game information, ensuring accurate and comprehensive search results.  
- **Responsive and User-Friendly Design**: Designed for a smooth and engaging experience across all devices, enhancing accessibility for users.  

## Technologies Used

- **Angular**: For building a responsive and interactive front-end with reusable components. 
- **Twitch's IGDB API**: To fetch real-time game data for search and discovery features.  
- **CSS Modules**: For styling components with scoped and maintainable CSS.
- **Figma**: Used for designing the website layout and creating a visually engaging user interface.
